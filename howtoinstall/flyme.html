SHV-E210S

1. Wipe System, Data, Dalvik Cache, Cache[시스템, 데이터, 달빅 캐시, 캐시 초기화]
2. Flash ROM[롬 설치]
3. Flash SKT RIL PATCH[SKT 통신패치 설치]
4. Reboot[재부팅]
5. settings-networks-APNs-SKT LTE
6. Reboot[재부팅]
7. Dial[다이얼] > *#*#4636#*#* > Phone information > set preferred network type: > LTE/GSM/CDMA auto(PRL)
8. Install google installer[구글 인스톨러 설치]
